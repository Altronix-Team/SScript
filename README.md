# SScript 
SScript is a project made to extend HScript and make it easier to use. It supports almost all of the Haxe structures.

Things done so far:
- [x] It's own class
- [x] Import
- [x] Public and Private
- [x] Final declaration
- [x] Inline variables and functions
- [x] Private access
- [x] Advanced call
- [x] Null Coalescing
- [x] Package
- [x] Dynamic Functions
- [x] True type declaration
- [ ] Class (In progress)
- [ ] Enum
- [ ] Abstract
